12 SWING IMPLEMENTATION GUIDE
Java Swing is fully capable of producing modern, premium UIs when used correctly. The techniques below
bridge the gap between standard Swing defaults and the design specified above.
Custom JPanel Painting
• Override paintComponent(Graphics g) — always call
super.paintComponent(g) first
• Cast to Graphics2D g2 = (Graphics2D) g
• Set RenderingHints.KEY_ANTIALIASING,
VALUE_ANTIALIAS_ON for smooth shapes
• Set RenderingHints.KEY_TEXT_ANTIALIASING,
VALUE_TEXT_ANTIALIAS_ON for smooth text
• Use g2.setRenderingHints(Map) to set all hints at once
Rounded Corners
• g2.fill(new RoundRectangle2D.Double(x, y, w, h, radius,
radius))
• For cards: radius 12–16px. For buttons: 8px. For pills:
full radius (h/2)
• Draw border separately: g2.setColor(borderColor);
g2.draw(roundRect)
• Use BasicStroke for border thickness control
Drop Shadows
• Custom ShadowBorder class using BufferedImage with
Gaussian blur (ConvolveOp)
• Or: draw 2–3 layers of rounded rects at decreasing
alpha before the main card
• Example: alpha 0.06, 0.04, 0.02 at offsets (2,4), (4,8),
(6,12)
• Add to JComponent via setBorder(new
ShadowBorder())
Smooth Animations
• Use javax.swing.Timer for all animations — never
Thread.sleep on EDT
• Easing: implement EasingUtils.easeOutCubic(t) = 1 -
pow(1-t, 3)
• For card hover scale: store float scale=1.0, animate
1.0→1.02 on mouseEnter
• Progress bars: animate value from 0→target over
800ms using Timer + easing
• Fade-in panels: use setOpaque(false), override paint to
